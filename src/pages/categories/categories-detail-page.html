Learn Enough JavaScript to Be Dangerous

Write Programs, Publish Packages, and Develop Interactive Websites with JavaScript

Author: Michael Hartl

Publisher: Addison-Wesley

Year: 2022

Pages: 304

Rating: 4

ISBN13: 9780137843749

JavaScript plays a key role in modern software development, not only because it is the only language that runs inside virtually all web browsers, but also because it has become widely used for back-end and general-purpose development as well. Although JavaScript is a big language, you dont need to learn every corner to start building useful things—this book focuses on the essentials youll actually use.
